#  Titanic Survival Analysis

##  Description
This project analyzes the Titanic dataset and builds a machine learning model to predict passenger survival.

##  Goals
- Explore the dataset (EDA)
- Identify key survival factors
- Build and evaluate ML models

## Key Findings
- Women survived more often than men
- 1st class passengers had higher survival rates
- Children had better chances to survive

##  Models
- Logistic Regression (~80%)
- Random Forest (~82%)

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
